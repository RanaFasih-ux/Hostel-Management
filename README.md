# Hostel Management System in C++

A C++ hostel management system using structured programming. Manages student records with add, delete, update, and search functions. Handles fee payments, generates reports, and stores data in text and binary files. Uses arrays, dynamic memory, and multiple functions for modular design.

## Features

- Add, Delete, Update, and Search Students
- Pay Fees and Track Payment Status
- Display All Students
- Generate Reports (text and binary)
- Data stored in text (hostel.txt) and binary (hostel.bin) files

## How to Compile & Run

1. Open terminal or command prompt.
2. Navigate to the project folder.
3. Compile the code:
   ```bash
   g++ Hostel.cpp -o Hostel
   ```
